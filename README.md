🍹 Mocktail Landing Page

A modern and responsive mocktail landing page UI concept designed for cafés, beverage brands, juice bars, and summer drink promotions.

## ✨ Features

- Elegant UI Design
- Responsive Layout
- Premium Beverage Showcase
- Clean Typography
- Refreshing Color Palette
- Call-to-Action Buttons
- Minimal and Modern Interface

## 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Setup

Clone the repository:

```bash
git clone https://github.com/your-username/mocktail-landing-page.git
```

Open the project folder:

```bash
cd mocktail-landing-page
```

Run the project by opening `index.html` in your browser.
